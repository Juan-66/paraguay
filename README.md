# **Base de Datos sobre Vacunados contra COVID-19 en Paraguay**

Este repositorio contiene un conjunto de datos relacionados con la campaña de vacunación contra el COVID-19 en Paraguay, orientado al análisis estadístico y epidemiológico.  
**Nota importante:** Los datos aún no están anonimizados, por lo que su acceso está restringido a fines educativos y de investigación.

---

## **Contenido**
El repositorio incluye:  
- Información sobre las dosis administradas por:  
  - **Fecha**  
  - **Región geográfica**  
  - **Tipo de vacuna**  
- Estadísticas generales:  
  - **Edad**  
  - **Género**  
  - **Rangos etarios**  
- **Aviso:**  
  Los datos aún no han sido anonimizados. Es fundamental evitar el uso indebido para proteger la privacidad de las personas involucradas.

---

## **Requisitos y Responsabilidad del Usuario**
El uso de esta base de datos está sujeto a las siguientes condiciones:  
1. **Fines permitidos:**  
   Exclusivamente para investigación educativa o académica.  
2. **Restricciones:**  
   Prohibido utilizar los datos reales sin consentimiento para evitar comprometer la privacidad de los involucrados.  
3. **Cumplimiento legal:**  
   El uso indebido de esta información puede acarrear consecuencias legales y éticas.
